# 功能说明 & 测试用例

## 1. Today 主页（HomeScreen）

### 功能
- 显示当日日期和欢迎语
- 每周习惯打卡网格（周一到周日）
- 上半部分最多显示 5 个习惯，超出可滚动
- 下半部分完整习惯列表，点击卡片可打卡/取消打卡
- 齿轮按钮 → 设置背景
- 加号按钮 → 新增习惯

### 测试用例
```
TC-HOME-01: 默认加载显示当日日期和欢迎语
TC-HOME-02: 打卡网格显示本周日期和完成状态
TC-HOME-03: 点击习惯卡片切换打卡状态（勾选/取消）
TC-HOME-04: 上半部分 >5 个习惯时内部可滚动
TC-HOME-05: 下半部分 LazyColumn 正常滚动
TC-HOME-06: 点击齿轮按钮弹出背景设置
TC-HOME-07: 点击加号导航到新增习惯页
```

---

## 2. 日历页（CalendarScreen）

### 功能
- 月历视图，显示当前月份的日期
- 有打卡记录的日期显示标记（△ 正常 / ★ 补签）
- 透明背景穿透

### 测试用例
```
TC-CAL-01: 加载时显示当前月份日历
TC-CAL-02: 有打卡记录的日期显示 △ 或 ★
TC-CAL-03: 背景图片/颜色正确穿透显示
```

---

## 3. 统计页（StatsScreen）

### 功能
- 统计卡片列表：完成率、总天数、已打卡天数、连续天数
- 点击统计卡片弹出详情弹窗
- **左滑露出编辑按钮**：滑动手势→卡片左移→编辑图标出现
- 任何操作（点击其他卡片、点击空白、滚动列表）自动复位
- 透明背景穿透

### 测试用例
```
TC-STAT-01: 列表显示所有习惯的统计数据
TC-STAT-02: 点击卡片弹出详情弹窗
TC-STAT-03: 弹窗显示完整数据（创建日期、结束日期、持续天数等）
TC-STAT-04: 左滑习惯卡片露出编辑按钮
TC-STAT-05: 点击已露出的卡片复位（不打开详情）
TC-STAT-06: 点击其他卡片先复位再打开详情
TC-STAT-07: 滚动列表时自动复位
TC-STAT-08: 点击编辑图标跳转到编辑页
```

---

## 4. 习惯编辑页（AddEditHabitScreen）

### 功能
- 新增习惯 / 编辑已有习惯
- Emoji 选择器：8列网格，63个自律主题 Emoji
- 提醒时间设置（TimePicker）
- 结束日期设置（DatePicker）
- 周目标设置（数字滚轴选择器）
- **删除习惯**（仅编辑模式）：确认对话框 → 删除
- 标签与值使用不同字体区分

### 测试用例
```
TC-EDIT-01: 新增模式显示"New Habit"，编辑模式显示"Edit Habit"
TC-EDIT-02: 名称输入框正常输入文字
TC-EDIT-03: Emoji 网格可滚动，点击选中显示边框
TC-EDIT-04: 点击 Reminder 时间弹出 TimePicker，Clear 可清除
TC-EDIT-05: 点击 End date 弹出 DatePicker，Clear 可清除
TC-EDIT-06: 周目标滚轴选择器可滚动选择 0-31
TC-EDIT-07: Save 按钮保存数据并返回上一页
TC-EDIT-08: 编辑模式显示"Delete habit"按钮
TC-EDIT-09: 点击 Delete 弹出确认对话框
TC-EDIT-10: 确认删除后数据删除并返回上一页
TC-EDIT-11: 后端日期验证（不能早于今天）
```

## 5. 背景系统（BackgroundManager）

### 功能
- 支持图片背景（缩放/偏移调整）
- 支持纯色背景（多种预设颜色）
- 所有页面统一穿透显示
- 图片预览支持手势缩放和平移

### 测试用例
```
TC-BG-01: 选择图片 → 预览对话框弹出
TC-BG-02: 预览支持双指缩放（0.3x-3.0x）
TC-BG-03: 预览支持单指平移
TC-BG-04: Apply 后背景在所有页面显示
TC-BG-05: 选择纯色背景替换图片背景
TC-BG-06: 缩放和偏移在切换页面后保持
```

## 6. 数据持久化

### 功能
- Room 数据库存储习惯和打卡记录
- Hilt 注入 Repository
- fallbackToDestructiveMigration 自动处理数据库迁移

### 测试用例
```
TC-DATA-01: 新增习惯后重启 App 数据保留
TC-DATA-02: 打卡记录持久化
TC-DATA-03: 删除习惯后数据不再显示
TC-DATA-04: 编辑习惯后数据更新
```
