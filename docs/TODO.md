## P0 — 体验问题

### 1. 切换背景时内容颜色不匹配，阅读困难

- 需要根据背景亮度动态调整 UI 颜色（文字、卡片、图标等）
- 可能的方案：自动检测背景亮度，切换文字颜色主题

### 2. Habit 展示范围及顺序
- 当前习惯按 sortOrder 排序，但无自定义排序功能
- 需要支持：置顶/排序调整、归档（archived）vs 活跃（active）筛选
- 可能的方案：拖拽排序 + 归档状态字段

## P2 — UI 优化

### 5. 编辑页面布局优化

#### 5.1 Save 和 Delete 按钮同一高度
- 当前 Save 按钮（全宽 PrimaryButton）和 Delete（TextButton）上下分布
- 需要两者在同一行，视觉平衡

#### 5.2 Reminder / End date / Weekly goal 与值并排
- 当前标签在上、值在下，上下分布占用较多垂直空间
- 改为标签在左、值在右并排布局，节省空间
